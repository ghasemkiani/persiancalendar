The formal calendar in general use in Iran is the Solar Hijri Calendar, also called the Jalali Calendar. We call it here the Persian calendar.

The class com.ghasemkiani.util.SimplePersianCalendar is a subclass of java.util.GregorianCalendar, with the added functionality that it can set/get date in the Persian calendar system. This is not a true implementation for a PersianCalendar, because the date fields are not accessible via Calendar.get(Calendar.YEAR), etc. (Instead, the Persian year, month, and day are accessible using a special object of type com.ghasemkiani.util.DateFields.) Such a true implementation may appear in a future version, or it may become available via IBM's "International Components for Unicode for Java" (see http://oss.software.ibm.com/icu4j).

The algorithms for conversion between Persian and Gregorian calendar systems are taken from the "Calendar Math Add-In for Excel" by Kees Couprie. As Couprie states, Mohammad Tahani from the Computer Research Center of Islamic Sciences (CRCIS) of I.R. Iran has contributed to the development of the Persian calendar in that package. Their efforts are gratefully acknowledged.

If you find any errors, especially if you find out that I have made a mistake in translating Couprie's VB code into Java, please report it to me (ghasemkiani@yahoo.com).

Using SimplePersianCalendar is straightforward. For example, to convert the current date into Persian date, we can write:

	SimplePersianCalendar c = new SimplePersianCalendar();
	DateFields t = c.getDateFields();
	System.out.println(
		"Current date in the Persian calendar is: " +
		t.getYear() + "/" +
		(t.getMonth() + 1) + "/" +
		t.getDay() + ".");
