Persian Calendar

Version 1.0

com.ghasemkiani.util.SimplePersianCalendar is a subclass of java.util.GregorianCalendar that performs Persian/Gregorian date conversion. This is not a true implementation for a PersianCalendar, because the date fields are not accessible via Calendar.get(Calendar.YEAR), etc. Such a true implementation may appear in a future version, or it may become available via IBM's "International Components for Unicode for Java" (see http://oss.software.ibm.com/icu4j).

The algorithms for conversion between Persian and Gregorian calendar systems are taken from "Calendar Math Add-In for Excel" by Kees Couprie. Couprie has expressed Special thanks to Mohammad Tahani from the Computer Research Center of Islamic Sciences (CRCIS) of I.R.Iran. I thank both Couprie and Tahani cordially.

I may have done some mistakes in translating Couprie's VB code into Java. If you find any errors, please report it to me (ghasemkiani@yahoo.com).

Using SimplePersianCalendar is strightforward. For example, to convert the current date into Persian date, we can write:

	SimplePersianCalendar c = new SimplePersianCalendar();
	DateFields t = c.getDateFields();
	System.out.println(
	"Current date in the Persian calendar is: " +
		t.getYear() + "/" +
		(t.getMonth() + 1) + "/" +
		t.getDay() + ".");
